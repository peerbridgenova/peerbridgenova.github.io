# PeerBridge Site

Static website for PeerBridge built with plain HTML, CSS, and JavaScript.

## Files

- `index.html` - homepage
- `about.html` - about page
- `articles.html` - articles page
- `team.html` - team page
- `contact.html` - contact page
- `styles.css` - shared styling
- `script.js` - shared interactions

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the root of that repo.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` and `/root`
5. Save.

Your site will go live at a GitHub Pages URL after deployment finishes.
